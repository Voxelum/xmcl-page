# Pierwsze kroki z lokalizacją

## Wymagania wstępne

- [Git](https://git-scm.com/). **Musisz** zainstalować Git, aby rozpocząć.
- [VSCode](https://code.visualstudio.com/). Wysoce zalecany edytor dla tłumaczy. Posiada zintegrowane narzędzie i18n do wyszukiwania kluczy tłumaczeń.
- [Node.js](https://nodejs.org/). Wymagany do lokalnego testowania tłumaczeń (zainstaluj najnowszą wersję).
- [pnpm](https://pnpm.io/installation). Wymagany do lokalnego uruchamiania i testowania projektu.

## Pierwsze kroki

### Fork i Clone <Badge type="danger">Wymagane</Badge>

Musisz wykonać fork i sklonować projekt:

```bash
git clone --recurse-submodules https://github.com/twoj-id/x-minecraft-launcher
```

### Instalacja <Badge type="danger">Wymagane</Badge>

Zainstaluj zależności projektu przy użyciu pnpm:

```bash
pnpm install
```

> Jeśli używasz `corepack`, możesz uruchomić `corepack enable .`, aby aktywować pnpm.

Jeśli napotkasz błędy o braku modułów, upewnij się, że projekt został sklonowany z opcją `--recurse-submodules`. Alternatywnie wykonaj:

```bash
git submodule update --init --recursive
```

i spróbuj ponownie uruchomić `pnpm install`.

### Znajdź plik lokalizacji

Pliki lokalizacji YAML znajdują się pod ścieżkami:

- `xmcl-keystone-ui/locales`
- `xmcl-electron-app/main/locales`

Nazwa pliku odpowiada kodowi języka (np. `pl.yaml` dla języka polskiego).

### VSCode z rozszerzeniem i18n-ally <Badge type="tip">Zalecane</Badge> <Badge type="info">Opcjonalnie</Badge>

Zainstaluj rozszerzenie `i18n-ally` (`lokalise.i18n-ally`).

W zakładce `PROGRESS` zobaczysz postęp tłumaczenia oraz liczbę brakujących kluczy. Możesz edytować tłumaczenia bezpośrednio w edytorze.

### Dodawanie nowego języka <Badge type="info">Opcjonalnie</Badge>

Jeśli dodajesz nowy język, dopisz go także do `assets/locales.json`.

Przykład dla języka francuskiego (`fr`):

```json
{
  "zh-CN": "简体中文",
  "zh-TW": "繁體中文",
  "en": "English",
  "ru": "Русский язык",
  "es-ES": "Español",
  "fr": "French"
}
```

Następnie utwórz odpowiedni plik **yaml** z kodem języka (np. `fr.yaml`).
