# Захаванне даных

Даныя XMCL падзяляюцца на дзве часткі:

1. Кэш і база даных XMCL, створаныя chromium
2. Даныя, звязаныя з Minecraft

## Кэш і база даных XMCL

Гэтыя даныя захоўваюцца ў сістэмнай папцы AppData, шлях да якой адрозніваецца ў залежнасці ад платформы.

::: code-group
```cmd [Windows]
%AppData%\xmcl
```
```cmd [Windows (APPX/appinstaller)]
# Версія < 0.34
%LocalAppData%\Packages\XMCL_ncdvebj03zfcm\LocalCache\Roaming\xmcl
# Версія >= 0.34 і < 0.40
%LocalAppData%\Packages\XMCL_68mcaawk44tpj\LocalCache\Roaming\xmcl
```
```sh [macOS]
~/Library/Application Support/xmcl
```
```sh [Linux]
~/.config/xmcl
```
:::

:::warning Заўвага
Не выдаляйце файлы адсюль, калі не ўпэўнены ў сваіх дзеяннях.
:::

Тут захоўваюцца розныя файлы канфігурацыі ў фармаце `json`, а таксама база даных рэсурсаў.

- **Даныя карыстальнікаў**: Акаўнты, спасылкі на скіны і г.д. Захоўваюцца ў файле `/user.json`.
- **Глабальныя налады**: Мова, проксі, крыніца загрузкі і г.д. Захоўваюцца ў файле `/settings.json`.
- **Кэш профіляў**: Шлях да апошняга абранага профілю і спіс усіх вядомых профіляў. Захоўваецца ў файле `/instances.json`.
- **Кэш Java**: Шляхі да выяўленых версій Java і іх метаданыя. Захоўваецца ў файле `/java.json`.
- **База даных рэсурсаў**: Метаданыя модаў, тэкстур і г.д. Захоўваюцца ў фармаце `leveldb` у папцы `/resources-v2`.
- **Логі (Журналы)**: Гісторыя логаў працы лаўнчара. Захоўваецца ў папцы `/logs`.

## Даныя Minecraft

Структура дырэкторыі Minecraft у XMCL трохі адрозніваецца ад стандартнай для аптымізацыі месца на дыску:

```sh
"Агульная папка даных"
└─ 📂mods # Агульная папка модаў для ўсіх профіляў
  └─ modA.jar # Файл мода, на які профілі ствараюць жорсткія спасылкі.
├─ 📂resourcepacks # Агульная папка тэкстур
├─ 📂shaderpacks # Агульная папка шэйдараў
├─ 📂versions # Агульныя версіі гульні
├─ 📂assets # Агульныя актывы Minecraft
├─ 📂libraries # Агульныя бібліятэкі
└─ 📂instances # Утрымлівае створаныя вамі профілі гульні
```

Большая частка змесціва аналагічная стандартнаму Minecraft, але папка `instances` змяшчае ізаляваныя файлы кожнага вашага створанага профілю гульні.
