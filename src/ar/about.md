إذا كان لديك winget، يمكنك استخدامه للتثبيت:

```bash
winget install CI010.XMinecraftLauncher
```

تثبيت HomeBrew متاح أيضاً عبر tap:

```bash
brew tap voxelum/xmcl
brew install --cask --no-quarantine voxelum/xmcl
```
