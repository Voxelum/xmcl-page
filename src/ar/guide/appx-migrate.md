# ترحيل AppX

ستوجهك هذه الصفحة إلى كيفية ترحيل بياناتك من تثبيت appx إلى تثبيت zip.

الفكرة بسيطة: فقط انسخ مجلد AppData xmcl الخاص بتطبيق AppX إلى موقع AppData العام.

> [!IMPORTANT]
> لا تحذف تطبيق إصدار AppX أثناء عملية الترحيل.

## ابحث عن بيانات AppX الخاصة بك

انتقل إلى المسار التالي للعثور على بيانات AppX الخاصة بك:

```cmd [Windows (APPX/appinstaller)]
# الإصدار < 0.34
%LocalAppData%\Packages\XMCL_ncdvebj03zfcm\LocalCache\Roaming\xmcl
# الإصدار >= 0.34 و < 0.40
%LocalAppData%\Packages\XMCL_68mcaawk44tpj\LocalCache\Roaming\xmcl
```

::: tip
استخدم مربع الإدخال (للتوجيه) في مستكشف الملفات (File Explorer) للانتقال إلى المسار المذكور أعلاه.
:::

## انسخ البيانات إلى AppData العام

انسخ مجلد `xmcl` بأكمله من الخطوة السابقة إلى:

```cmd [Windows]
%AppData%\xmcl
```

## ابدأ تشغيل XMCL الجديد واحذف القديم

بعد نسخ البيانات، يمكنك بدء تشغيل XMCL الجديد من ملف zip.

بمجرد التأكد من أن XMCL الجديد يعمل بشكل صحيح، يمكنك حذف تطبيق XMCL القديم.

::: tip
سيؤدي حذف تطبيق AppX إلى إزالة البيانات القديمة **تلقائياً**، لذا لا داعي للقلق بشأن البيانات المتبقية.
:::
