# Початок роботи з локалізацією

## Попередні вимоги

- [Git](https://git-scm.com/). Для початку роботи **обов'язково** встановіть Git.
- [VSCode](https://code.visualstudio.com/). Рекомендований редактор для перекладу. Він має вбудоване візуальне розширення для пошуку ключів перекладу.
- [Node.js](https://nodejs.org/). Необхідний для тестування перекладу локально (встановіть останню версію).
- [pnpm](https://pnpm.io/installation). Необхідний для встановлення залежностей та запуску сайту локально.

## Як розпочати

### Форк та клонування <Badge type="danger">Обов'язково</Badge>

Вам потрібно створити форк репозиторію на GitHub та клонувати його за допомогою Git:

```bash
git clone --recurse-submodules https://github.com/your-id/x-minecraft-launcher
```

### Встановлення <Badge type="danger">Обов'язково</Badge>

Вобіть встановлення залежностей проекту за допомогою pnpm (або скористайтеся corepack).

Перейдіть до клонованої папки `x-minecraft-launcher` та виконайте команду:

```bash
pnpm install
```

> Якщо ви використовуєте `corepack`, виконайте `corepack enable .` для активації pnpm.

Якщо виникає помилка про те, що модуль не знайдено, переконайтеся, що ви клонували проект із параметром `--recurse-submodules`. Якщо ви забули це зробити, виконайте:

```bash
git submodule update --init --recursive
```

а потім знову запустіть `pnpm install`.

### Пошук файлу локалізації

Файли перекладів знаходяться за шляхами:

- `xmcl-keystone-ui/locales`
- `xmcl-electron-app/main/locales`

Ім'я файлу відповідає коду мови. Ви можете орієнтуватися на стандарти кодів мов. Наприклад, `uk.yaml` для української.

### Використання VSCode з розширенням i18n-ally <Badge type="tip">Рекомендовано</Badge> <Badge type="info">Необов'язково</Badge>

Встановіть розширення `i18n-ally` (`lokalise.i18n-ally`).

У вкладці `PROGRESS` цього розширення ви побачите поточний прогрес перекладу та кількість відсутніх ключів для вибраної мови. Ви можете редагувати переклад прямо в інтерфейсі розширення.

### Додавання нової мови <Badge type="info">Необов'язково</Badge>

Якщо ви додаєте абсолютно нову мову, додайте код та назву мови до файлу `assets/locales.json`.

Наприклад, якщо ви хочете додати французьку (`fr`), відкрийте файл `assets/locales.json`:

```json
{
  "zh-CN": "简体中文",
  "zh-TW": "繁體中文",
  "en": "English",
  "ru": "Русский язык",
  "es-ES": "Español",
  "fr": "French"
}
```

Потім створіть відповідний **yaml**-файл із кодом локалі. Наприклад, `fr.yaml`.
