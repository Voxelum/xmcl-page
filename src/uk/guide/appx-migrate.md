# Міграція AppX

Ця сторінка допоможе вам перенести дані з версії AppX у версію Zip.

Ідея проста: скопіювати папку xmcl з AppData для AppX у загальне розташування AppData.

> [!IMPORTANT]
> **НЕ** видаляйте версію AppX під час міграції.

## Знайдіть дані AppX

Перейдіть за таким шляхом, щоб знайти свої дані AppX:

```cmd [Windows (APPX/appinstaller)]
# Версія < 0.34
%LocalAppData%\Packages\XMCL_ncdvebj03zfcm\LocalCache\Roaming\xmcl
# Версія >= 0.34 та < 0.40
%LocalAppData%\Packages\XMCL_68mcaawk44tpj\LocalCache\Roaming\xmcl
```

::: tip
Використовуйте поле введення шляху у Провіднику (File Explorer), щоб перейти за вказаним вище шляхом.
:::

## Скопіюйте дані у загальну папку AppData

Скопіюйте всю папку `xmcl` з попереднього кроку до:

```cmd [Windows]
%AppData%\xmcl
```

## Запустіть новий XMCL та видаліть старий

Після копіювання даних ви можете запустити новий XMCL з архіву zip.

Коли переконаєтеся, що новий XMCL працює належним чином, можете видалити стару програму XMCL.

::: tip
Видалення програми AppX **автоматично** видалить старі дані, тому вам не потрібно турбуватися про залишки даних.
:::
