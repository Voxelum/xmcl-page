# Multiplayer Guide

XMCL provides built-in support for multiplayer gaming through P2P (Peer-to-Peer) connections.

## P2P Multiplayer

XMCL has a built-in P2P multiplayer feature that allows you to play with friends without needing to set up a server.

### How to Use P2P

1. **Create a room**
   - Launch your Minecraft instance
   - Click on the P2P button in the multiplayer menu
   - Create a new room

2. **Share the room code**
   - Share the room code with your friends
   - They can join using the same P2P menu

3. **Play together**
   - Once your friends join, you can play together
   - No server setup required!

### Requirements

- All players must have XMCL installed
- All players must have the same Minecraft version and mods
- Internet connection is required

## Traditional Multiplayer

You can also connect to traditional Minecraft servers:

1. Click **Multiplayer** in the main menu
2. Click **Add Server**
3. Enter the server address
4. Click **Done** and connect

### Popular Servers

- Hypixel: `mc.hypixel.net`
- Mineplex: `us.mineplex.com`
- CubeCraft: `play.cubecraft.net`

## Troubleshooting

### Can't Connect to P2P

- Check your internet connection
- Make sure all players have the same version
- Try restarting XMCL

### Can't Connect to Server

- Verify the server address
- Check if the server is online
- Make sure your firewall isn't blocking the connection
